VIPER-SWIFT
===========

An example Todo list app written in Swift using the VIPER architecture.

### 实战
添加Login登录模块
#### 分析VIPER设计顺序

V：输入用户名和密码，验证用户信息，
E：用户名，密码结构
I：实体数据结构层处理，分析业务逻辑
P：UI校验逻辑，UI登录调整逻辑
R：线框+路由控制跳转逻辑
